This mod contains a more than 90 signs models based on Brazil Road Signs. It might be used together Addresses Mod and Write Everywhere to work.

(Em português no fim da página!)

## How to make it available on Addresses Mod

- Firstly ensure that the three mods are active and loaded
- Go to Write Everywhere button, at top left of the screen
- Go to the Prefab Templates Setup
- Find Addresses Mod entry on the list
- Choose one of the custom slots (1 to 7) and select this mod on first dropdown, then "Base" on second dropdown

After done that, it might be shown in the Addresses mod while placing or editing road markers, instead of the "Custom X" label on content dropdown.

It will also show some additional controls, like:
- Signs group: A (yellow), R (White/red), Horizontal and Vertical signs
- Inside each group, there are the signs available. For A and R groups, they are sorted as they are in the Brazilian Traffic Code (CTB), so you can find them easily if you know it.
- Additionally, an extra dropdown may be shown to select a variation of the sign, like the "No Parking" sign, which has a variation for "No Stopping" and "Regulated Parking".
- The signs that are about some metric (weight, size, velocity) will follow the game metric system, so they will be shown in meters, tons, etc. or feet/yards/miles, pounds, etc.
  - These signs will display an extra input field to set the value of the metric, which will be used to display the sign correctly.
  - For metric system lengths and weights, the value will always be divided by 10, so a value of 100 will be shown as 10.0 m or 10.0 tons.
- Some signs have no parameters but are dynamic based on Addresses information, like Highway plates (State and Federal). As well for mileage marker model.


## About the future

Even initially being a Brazilian pack, it is planned to be expanded to have more signs from other countries by community suggestions and contributions. You can use the forums to send suggestions 
or files to be included in the pack. However, I don't guarantee when they will be added.


# Português!

Este mod contém mais de 90 modelos de placas baseadas nas placas de trânsito do Brasil. Ele pode ser usado junto com o Addresses Mod e o Write Everywhere para funcionar.

## Como torná-lo disponível no Addresses Mod
- Primeiro, certifique-se de que os três mods estão ativos e carregados
- Vá para o botão Write Everywhere, no canto superior esquerdo da tela
- Vá para a Configuração de Modelos de Prefab
- Encontre o item do Addresses Mod na lista
- Escolha um dos slots personalizados (1 a 7) e selecione este mod no primeiro menu suspenso, depois "Base" no segundo menu suspenso

Depois de fazer isso, ele deve ser exibido no mod Addresses ao colocar ou editar marcadores de estrada, em vez do rótulo "Personalizado X" no menu suspenso de conteúdo.

Ele também mostrará alguns controles adicionais, como:
- Grupo de placas: A (amarelo), R (branco/vermelho), placas horizontais e verticais
- Dentro de cada grupo, estão as placas disponíveis. Para os grupos A e R, elas são organizadas como estão no Código de Trânsito Brasileiro (CTB), para que você possa encontrá-las facilmente se souber.
- Além disso, um menu suspenso extra pode ser exibido para selecionar uma variação da placa, como a placa "Proibido Estacionar", que tem uma variação para "Proibido Parar" e "Estacionamento Regulamentado".
- As placas que tratam de alguma métrica (peso, tamanho, velocidade) seguirão o sistema métrico do jogo, portanto serão exibidas em metros, toneladas, etc. ou pés/jardas/milhas, libras, etc.
  - Essas placas exibirão um campo de entrada extra para definir o valor da métrica, que será usado para exibir a placa corretamente.
  - Para comprimentos e pesos do sistema métrico, o valor sempre será dividido por 10, então um valor de 100 será exibido como 10.0 m ou 10.0 toneladas.
- Algumas placas não têm parâmetros, mas são dinâmicas com base nas informações do Addresses, como placas de rodovias (Estadual e Federal). Assim como o modelo de marco de milhagem.

## Sobre o futuro

Mesmo sendo inicialmente um pacote brasileiro, está planejado ser expandido para ter mais placas de outros países por sugestões e contribuições da comunidade. Você pode usar os fóruns para enviar sugestões
ou arquivos para serem incluídos no pacote. No entanto, não garanto quando eles serão adicionados.
